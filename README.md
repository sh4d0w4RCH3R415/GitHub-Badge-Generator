# GitHub Badge Generator
### DESCRIPTION
Will generate the link required for a GitHub README badge.

### HOW TO USE
 - To add a special character of type '\_' or '-', type 2 '\_' or '-'.
 - To add a normal whitespace, add 1 '\_'.
